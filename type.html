المستوى الصعب

ما هو الفرق بين 'type' و 'interface' في TypeScript؟

type is for complex types and interface is for simple types.
type is for defining new types, interface is for defining object shapes.
there is no difference

ما هو الفرق بين 'interface' و 'type' في تعريف الدوال في TypeScript؟

interface is used for defining shapes of objects and classes while type is used for defining more complex types.
type can be used to define object shapes and primitive types but interface cannot be used for primitive types.
there is no difference
